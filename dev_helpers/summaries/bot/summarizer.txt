This file, `summarizer.py`, provides functionality to summarize conversation messages using the Google Gemini API. It configures the API with a key from a `Config` object. The main function, `summarize`, takes a list of message strings, formats them into a Russian prompt for a user-therapist conversation, and sends it to the `gemini-2.5-pro-preview-05-06` model. It returns the generated summary, limited to a maximum of 5 sentences in the prompt, or an error message if the API call fails or returns an empty response.
