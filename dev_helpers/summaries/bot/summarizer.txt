This file, `summarizer.py`, provides functionality to summarize conversation messages using the Google Gemini API. It configures the API with a key from `config.py`. The main function, `summarize`, takes a list of message strings, formats them into a prompt (in Russian) requesting a concise summary, and sends it to the 'gemini-2.5-pro' model. It returns the generated summary text or an error message if the API call fails.
