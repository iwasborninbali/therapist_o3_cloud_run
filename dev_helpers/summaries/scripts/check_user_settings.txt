This Python script is designed to check and manage user settings stored in a Firestore database. It retrieves settings for a hardcoded user ID using `get_user_settings`. If no settings are found for that user, it sets a default timezone ("Asia/Makassar") using `set_user_settings` and then verifies the change. The script also adjusts the Python path to import the necessary Firestore client module from the project root.
