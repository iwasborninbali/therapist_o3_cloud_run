This Python script `test_proactive.py` is designed to test the proactive message functionality of a bot. It verifies user timezone settings and retrieves users with configured timezones from Firestore. The script tests the core logic (`should_send_proactive_message`) by temporarily setting the configured proactive hours (like `config.MORNING_HOUR`) to the user's current local hour to force a test condition. It logs whether a message should be sent and attempts to generate a test message.
